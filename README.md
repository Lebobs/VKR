# Программный комплекс Моделирование распространения загрязнителя в водной среде на основе модели "мелкой воды"

Данное программное обеспечение предназначено для численного моделирования и визуализации процессов распространения загрязняющих веществ в открытых водоемах на основе двумерной модели мелкой воды. Архитектура системы включает в себя вычислительное ядро на языке Python, реляционное хранилище данных под управлением СУБД PostgreSQL и технологический слой для интеграции модулей.

## Системные требования и зависимости

Для корректной работы комплекса требуется установленная среда выполнения Python версии 3.10 или выше, а также развернутый сервер баз данных PostgreSQL. Основными программными компонентами системы являются библиотека NumPy для проведения высокопроизводительных матричных вычислений, драйвер Psycopg2 для стабильного взаимодействия с базой данных и библиотека Matplotlib, отвечающая за генерацию графических тепловых карт распределения концентраций загрязнителей.

## Порядок развертывания и запуска

Перед первым запуском необходимо настроить параметры подключения к базе данных в конфигурационном файле и импортировать структуру таблиц из прилагаемого SQL-скрипта инициализации схемы. После успешного создания таблиц проектов, сценариев и результатов следует выполнить установку всех необходимых библиотек с помощью стандартного пакетного менеджера. Запуск веб-приложения осуществляется путем выполнения главного исполняемого файла в терминале, после чего графический интерфейс становится доступен по локальному адресу в веб-браузере.
